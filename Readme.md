# MOD Description
The Earth is empty after a great cataclysm. Humanity escaped to the stars. A fleet of generation ships has returned. You lead one of them.

# Using this MOD
Start a game with the following Game Rules:
- Set "Base Construction Capacity" to "Maximum".
- Set "AI Aggressivity" to "High".

The difficulty of the game will be scaled by the country selected. Countries in China and India are easier to handle due to the massive amount of subsistence farming available from the start. Countries in Australia and in northern climates are harder.

# Python Script Setup
Requires Python 3.12 and Pip
- python -m venv venv
- venv\Scripts\activate
- pip install python-liquid
